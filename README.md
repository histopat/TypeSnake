# TypeSnake
TypeSnake, React ve TypeScript kullanılarak geliştirilmiş basit ama eğlenceli bir yılan oyunudur.  
Oyunun amacı klasik yılan oyunundaki gibi elma yiyerek büyümek, fakat bu versiyonda ekstra bir kural var: **Mavi zehir** yendiğinde yılan bir parça kısalır. Eğer yılan zehirle tamamen yok olursa oyun biter.  

## Özellikler
- Yeşil yılan hareket eder ve yön tuşlarıyla kontrol edilir.
- Kırmızı elma yendiğinde yılan uzar.
- Mavi zehir yendiğinde yılan kısalır.
- Yılan kendine veya duvarlara çarptığında oyun biter.
- Oyun bittiğinde ekranda **Yeniden Başlat** butonu çıkar.
- Açık/Koyu tema desteği vardır, kullanıcı istediği zaman değiştirebilir.
- Şık CSS ile modern bir görünüm.

## Kurulum ve Çalıştırma
   ```bash
   git clone https://github.com/histopat/TypeSnake.git
   cd typesnake
   npm install
   npm start
```

## Teknolojiler
- React
- TypeScript
- HTML5 Canvas
- CSS ile tema desteği

## Gelecek Özellikler
- Skor sistemi ve skor tablosu
- Zorluk seviyeleri (hız artışı)
- Ses efektleri
- Mobil uyumlu kontroller
- Farklı zehir ve bonus türleri
